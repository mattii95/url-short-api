# SHORTEN URL API

1. Clone Project
2. Install dependencies
```
npm install
```
3. Clone the __.env.template__ file and rename it to __.env__.
4. Change environment variables
5. Raise DB
```
docker compose up -d
```
6. Start app
```
npm run start:dev
```

Project URL:  [url-short-api](https://github.com/mattii95/url-short-api)
